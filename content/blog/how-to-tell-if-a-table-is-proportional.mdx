Here's a table:

| $x$ | $y$ |
| :-: | :-: |
| 2 | 8 |
| 4 | 16 |
| 6 | 24 |
| 9 | 36 |

Is it proportional? You can answer in about 5 seconds.

## The Constant Ratio Method

Divide $y$ by $x$ for every row. If every division gives the same number, the table is proportional.

For the table above:

- $8 \div 2 = 4$
- $16 \div 4 = 4$
- $24 \div 6 = 4$
- $36 \div 9 = 4$

Every answer is 4. That means $k = 4$, and the relationship is $y = 4x$. **Proportional.**

## Try a Non-Proportional Table

| $x$ | $y$ |
| :-: | :-: |
| 1 | 5 |
| 2 | 9 |
| 3 | 13 |
| 4 | 17 |

Check the ratios:

- $5 \div 1 = 5$
- $9 \div 2 = 4.5$
- $13 \div 3 = 4.33$
- $17 \div 4 = 4.25$

The ratios change every time. **Not proportional.**

Notice the pattern: the $y$ values increase by 4 each time, and $x$ increases by 1. This follows $y = 4x + 1$ — a straight line, but shifted up by 1. That extra constant is what breaks proportionality.

## Why the Ratio Method Works

When two quantities are proportional, the ratio $y \div x$ stays constant by definition. That constant is $k$, the **constant of proportionality**, from the equation $y = kx$.

If the ratio ever changes, the relationship can't be written as $y = kx$, so it isn't proportional.

## Common Trap: Adding vs. Multiplying

Students often confuse "increases by the same amount" with "multiplies by the same factor."

In a proportional table, when $x$ doubles, $y$ doubles. When $x$ triples, $y$ triples. That's **multiplication by a constant factor**, not addition.

Look at this table:

| $x$ | $y$ |
| :-: | :-: |
| 0 | 0 |
| 1 | 3 |
| 2 | 6 |
| 3 | 9 |

$x$ increases by 1, $y$ increases by 3. But more importantly, $3 \div 1 = 3$, $6 \div 2 = 3$, $9 \div 3 = 3$. The ratio holds.

Now this table:

| $x$ | $y$ |
| :-: | :-: |
| 0 | 2 |
| 1 | 5 |
| 2 | 8 |
| 3 | 11 |

$x$ increases by 1, $y$ increases by 3. Same addition pattern. But $5 \div 1 = 5$, $8 \div 2 = 4$ — ratios are different. This is $y = 3x + 2$. The $+2$ breaks proportionality.

**The addition pattern looks the same. Only the ratio reveals the truth.**

## Step-by-Step Check

1. Pick any row (skip $0, 0$ if present — $0 \div 0$ is undefined).
2. Divide $y$ by $x$.
3. Repeat for another row.
4. If the results match, check a third row.
5. If all match, the table is proportional.

## FAQ

**What if the table has negative numbers?**
The same method works. Divide $y$ by $x$ for each row. If all ratios are the same (including negative), the table is proportional.

**What if a row has $x = 0$?**
Skip that row. $0 \div 0$ is undefined. But if $x = 0$ and $y \neq 0$, the relationship cannot be proportional (in $y = kx$, $y$ must be 0 when $x = 0$).

**Does the ratio need to be a whole number?**
No. $k$ can be a fraction, decimal, or even a negative number. The only requirement is consistency.

**How many rows do I need to check?**
At least two (where $x \neq 0$). Three is safer to rule out coincidence.

**Can you have a proportional table with a missing value?**
Yes. If you know the constant of proportionality $k$, you can find the missing value using $y = kx$.
